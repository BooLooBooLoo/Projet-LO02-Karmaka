L'application du jeu Karmaka se compose de 4 packages :
  • Karmaka.src où se trouve la base du jeu
  • Graphique où se trouve l'interface graphique réalisée sur le modèle MVC
  • Au sein de Graphique, on trouve le package State qui représente les différents composants de l'interface graphique réalisée avec Swing.
  • Cartes où se trouve les différentes cartes du jeu.
Réalisé par Ali MIKOU et Hoang-Viêt LE.
